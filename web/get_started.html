How to use Dataflow kit

  1. Enter web site address on the top to load a webpage you want to extract data from. Follow the next steps to extract data with Dataflow Kit.
  2. Start selecting elements on the loaded page in Web tab by clicking button on the left side.

    'Selection mode' will be turned on and the icon will be exchanged with

  3. Click on the first element on the page to be matched by selector. It will be highlighted in Green.

    A special rule (CSS Selector) for extracting the data will be generated.
    Dataflow Kit will suggest similar elements that you may also want to select and mark them in Yellow.

  4. Click on a highlighted element removes it from selection and turns an element's colour to Red.
  5. Otherwise, you can click on unhighlighted element to add it to the current selection.

    Iterative process of selection and rejection (steps 4 and 5) helps to specify more precisely the rules for data scraping.
    Holding 'shift' key while moving the mouse over selections will let you select elements inside them.

  6. Press button to finish selection process. Specified Selectors will be added to your Collection.

    18 The number in a circle near selector is the count of elements selected.

  7. Once you have selected all the data you want for your first selector, you can repeat steps from 2 to 6 to add more ones to the Collection.
  8. Click Data tab after selection of all necessary elements on the page.

    Now you can review extracted data and save it to either one of the following formats: CSV, JSON, Microsoft Excel or XML.
